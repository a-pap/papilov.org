Claude забывает всё между разговорами. Встроенная память генерируется автоматически, без структуры, и отстаёт на дни.

**Memex даёт структурированную память, которой управляешь ты** — приватный GitHub-репозиторий как единственный источник истины. Контекст живёт в человекочитаемых markdown-файлах, версионируется в git, подгружается по требованию. Один и тот же репозиторий — мозг для Claude.ai, Claude Code, мобильного приложения и фоновых задач.

## Что ты получаешь

- **Единый контекст на всех поверхностях.** Утром принял решение с телефона — днём Claude Code на ноутбуке уже знает о нём.
- **Человекочитаемое хранилище.** Markdown в git, а не непрозрачный SQLite. Каждое изменение — коммит, который можно прочитать, сравнить и откатить.
- **Поэтапная загрузка контекста.** ~3K токенов на старте, больше — только если тема этого требует. Никаких 50K токенов истории в каждом разговоре.
- **Полное восстановление после сбоя.** Свежий Claude без memory edits восстанавливает весь контекст из репозитория примерно за две минуты.
- **26 MCP-инструментов поверх Cloudflare Workers.** Опционально — для поверхностей, где нет git. Но ядро работает и без них, из одного репо.
